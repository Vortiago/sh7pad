# sh7pad user guides

Step-by-step instructions for the sh7pad browser editor. Each guide assumes the app is open at `/sh7pad/` and the disclaimer has been dismissed.

- [Create a design](create-design.md): start a new project (design or manual mode), rename it, delete it.
- [Preview a design](preview-design.md): switch to Preview, play and scrub, change speed and zoom, tune the Preview Settings panel.
- [Edit a design](edit-design.md): work with satin segments, flip a straight segment to satin, toggle density, add a tracing image, set thread tension.
- [Place stitches in Manual Mode](manual-mode.md): use Add + Needle / Jump / Satin to author a design one stitch at a time, read the live click window, and work with the Start Lock.
- [Start Stitch and Carriage Start](start-stitch.md): move the foot icon and the green diamond that mark where the carriage parks and where the first needle drops.
- [Export a design](export-design.md): save a project to `.sh7` or `.sh7c.json`.
- [Import a design](import-design.md): load an existing `.sh7`, `.sh7c`, or `.sh7c.json` file.

## A note on storage

Projects live in your browser (IndexedDB). They survive reload but are tied to this browser profile and this origin. Use Export to keep a copy off-device.
